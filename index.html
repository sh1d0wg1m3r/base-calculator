1. The "Any-to-Any" Conversion Strategy

The safest way to convert between two different bases (e.g., Base 5 to Base 7) is to use Base 10 as a bridge.

The Golden Rule:
Source Base $\rightarrow$ Convert to Base 10 $\rightarrow$ Convert to Target Base

Part A: Any Base to Base 10 (Multiplication)

Use Positional Notation. Multiply each digit by the base raised to the power of its position.

Example: Hex 2C to Decimal

Position:   1   0
Digits:     2   C (12)
Calculation: (2 × 16¹) + (12 × 16⁰)
           = 32 + 12 
           = 44

Part B: Base 10 to Any Base (Division)

Use the Division-Remainder Method.

  1. Divide decimal number by target base.
  2. Write down remainder.
  3. Divide quotient by base again.
  4. Repeat until quotient is 0.
  5. Read remainders Bottom-to-Top.

Example: Decimal 13 to Binary

  • $13 \div 2 = 6$ R 1
  • $6 \div 2 = 3$ R 0
  • $3 \div 2 = 1$ R 1
  • $1 \div 2 = 0$ R 1
  • Result: 1101

2. Binary Arithmetic (Base 2)

Addition Rules

$0+0=0$, $0+1=1$, $1+1=10$ (Carry 1), $1+1+1=11$ (Carry 1)

  1011  (11)
+ 0110  (6)
------
 10001  (17)

3. Octal Arithmetic (Base 8)

Addition Logic

Digits 0-7. If sum $\ge$ 8, subtract 8 and carry 1.

   145 (Octal)
+   63
------
   230
1. 5+3=8 -> 0 carry 1
2. 4+6+1=11 -> 3 carry 1 (11-8=3)

4. Hexadecimal Arithmetic (Base 16)

Addition Logic

Digits 0-9, A-F. If sum $\ge$ 16, subtract 16 and carry 1.

   1A (26)
+  2C (44)
-----
   46 (70)
1. A(10)+C(12)=22. 22-16=6 carry 1.
