1. The "Any-to-Any" Conversion Strategy

The safest way to convert between two weird bases (e.g., Base 5 to Base 7) is to use Base 10 as a bridge.

The Algorithm:
Source Base â†’ Convert to Base 10 â†’ Convert to Target Base

Part A: Any Base to Base 10

Use Positional Notation. Multiply each digit by the base raised to the power of its position (starting from 0 on the right).

Example: Convert Hex 2C to Decimal

  • Digits: 2, C (where C = 12)
  • Positions: 1, 0
  • Calculation: $(2 \times 16^1) + (12 \times 16^0)$
  • $32 + 12 = 44$

Part B: Base 10 to Any Base

Use the Division-Remainder Method.

  1. Divide the number by the target base.
  2. Write down the remainder.
  3. Take the quotient (result) and divide it by the base again.
  4. Repeat until the quotient is 0.
  5. Read remainders from bottom to top (last remainder is the first digit).

Example: Convert Decimal 13 to Binary

  • $13 \div 2 = 6$ (Remainder 1)
  • $6 \div 2 = 3$ (Remainder 0)
  • $3 \div 2 = 1$ (Remainder 1)
  • $1 \div 2 = 0$ (Remainder 1)
  • Result (Bottom-up): 1101

2. Binary Arithmetic (Base 2)

Only digits 0 and 1 allowed. Remember: $1 + 1 = 10$ (2 in decimal).

Binary Addition (+)

Rules: $0+0=0$, $0+1=1$, $1+1=10$ (write 0, carry 1), $1+1+1=11$ (write 1, carry 1).

  1011  (11)
+ 0110  (6)
------
 10001  (17)

Binary Subtraction (-)

Rules: $0-0=0$, $1-1=0$, $1-0=1$. For $0-1$, borrow from the left (the 0 becomes 10 (which is 2)).

  1101  (13)
- 1010  (10)
------
  0011  (3)

Binary Multiplication (*)

Same as decimal, but easier! You either multiply by 1 (copy the number) or 0 (write zeros).

    101 (5)
  x  11 (3)
  -----
    101 (1 * 101)
+ 1010  (1 * 101 shifted left)
------
  1111  (15)

Binary Division (/)

Long division. "Does the divisor fit?" Yes = 1, No = 0.

3. Octal Arithmetic (Base 8)

Digits 0-7. If you hit 8, you carry over.

Octal Addition

Example: $7 + 1 = 10_8$. $6 + 4 = 12_8$ (because $6+4=10$ dec, and $10 = 1 \times 8 + 2$).

   145 (Octal)
+   63
------
   230
   
1. 5+3=8. 8 is not in Octal. 8 = 10 (base 8). Write 0, carry 1.
2. 4+6+1(carry) = 11 dec. 11 = 8+3 = 13 (base 8). Write 3, carry 1.
3. 1+1(carry) = 2.

4. Hexadecimal Arithmetic (Base 16)

Digits 0-9 and A-F (A=10, B=11, C=12, D=13, E=14, F=15).

Hex Addition

Example: $A + 1 = B$. $F + 1 = 10_{16}$.

   1A (26 dec)
+  2C (44 dec)
-----
   46 (70 dec)

1. A(10) + C(12) = 22 dec. 
   22 = 16 + 6 = 16(16). Write 6, carry 1.
2. 1 + 2 + 1(carry) = 4.
